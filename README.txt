# ej1_eras

Se pide crear un sistema para la compra de boletos de un concierto. 
Se solicita que el sistema tenga un menú, donde aparezcan las opciones de: Nuevo comprador, Nueva solicitud de boletos, Consultar disponibilidad total, Consultar disponibilidad individual, Reporte de caja, Código especial, Salir. 

Para la compra de boletos, se deben solicitar varios datos al usuario (nombre, email, cantidad de boletos a comprar y presupuesto máximo.), asimismo, se pide realizar un sistema aleatorio de selección de números, para así regular la venta de boletos. De igual manera, se debe validar si las localidades en las que se pretende comprar tienen espacio disponible y si el comprador tiene el presupuesto para pagarla. Finalmente, si alguna persona tiene un código especial, se le priorizará su entrada y tendrá acceso a la mejor localización, con un precio mayor (el código especial debe pertenecer a la secuencia de Fibonacci).
